Component: ccmBreadcrumb

Breadcrumb navigation showing current page location in site hierarchy.

Uses an ordered list inside a navigation landmark, with separators hidden
from screen readers. Links are rendered with ccmButton[variant="link"].
Optionally injects JSON-LD (schema.org BreadcrumbList) for SEO.

Category
navigation
Standards
all-10

Prop: is

Root element tag (navigation landmark recommended).

Prop: items

Ordered list of breadcrumb items. Last item is treated as current page.

Prop: size

Control link size; forwarded to ccmButton.

Prop: variant

Visual variant: default (no wrap) or wrap (allow line wrap).

Prop: separator

Separator style: chevron, slash, or custom via #separator slot.

Prop: iconName

Icon name for chevron separator, used with .icon utility.

Prop: itemPaddingInline

Inline padding for link items (space token key).

Prop: ariaLabel

Aria-label for the nav landmark.

Prop: includeJsonLd

Include schema.org JSON-LD BreadcrumbList in document head.

Prop: baseUrl

Absolute base URL for JSON-LD items (e.g., https://example.com).
